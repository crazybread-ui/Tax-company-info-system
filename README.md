1. Информационно-справочная система страховой компании

Веб-приложение для автоматизации процессов управления страховыми полисами, клиентами и страховыми случаями.
Система предоставляет инструменты для регистрации пользователей, оформления полисов, ведения клиентской базы, учёта заявлений о страховых случаях и анализа статистики.
Разработана в рамках курсовой работы по дисциплине «Современные технологии программирования» в Финансовом университете при Правительстве Российской Федерации.

2. Функциональные возможности

2.1. Аутентификация и авторизация

Регистрация и вход с валидацией данных и шифрованием паролей (BCrypt).

Ролевая модель пользователей:

USER — подача заявлений, просмотр информации о полисах и статусах;

ADMIN — управление клиентами, полисами, типами полисов и пользователями.

Защита от CSRF-атак и несанкционированного доступа.

2.2. Управление клиентами и полисами

CRUD-операции для клиентов, страховых полисов и типов полисов.

Фильтрация и сортировка данных по ФИО, номеру полиса, типу страхования и дате оформления.

Динамическая работа с базой данных с использованием JPA Specifications.

2.3. Управление страховыми случаями

Регистрация и обработка страховых заявлений.

Назначение статусов заявлений (в обработке, одобрено, отклонено).

Просмотр истории страховых случаев по клиентам и полисам.

2.4. Административный модуль

Управление пользователями (назначение ролей, удаление и блокировка аккаунтов).

Защита от самовольного изменения роли администратора.

Контроль корректности введённых данных.

2.5. Статистика

Визуализация количества страховых заявлений по месяцам и типам полисов.

Анализ активности клиентов и распределения полисов по категориям.

2.6. Пользовательский интерфейс

Адаптивные шаблоны на основе Thymeleaf и Bootstrap 5.

Динамические компоненты: фильтры, таблицы, формы ввода, сортировка.

Интуитивно понятная структура навигации и интерфейса.
